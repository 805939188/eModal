Alert


eModal.alert(), this launch a non blocking alert, using a bootstrap modal!


Alert modal

An amazing message!

How to use

Simple way - Default title
var message = "Wello World!";
                         
eModal.alert(message);
Simple way - Custom title
var url = "Wello World!";
var title = "Alert Title";
                         
eModal.ajax(message, title);
Tip: You can use DOM elements as messages!
Message using DOM element
var options = {
        message: $("#my-element-id"),
        title: 'Header title',
        size: 'lg',
        subtitle: 'smaller text header',
        useBin: true
        ...
    };
                 
eModal.alert(options);
The element whit id 'my-element-id' will be moved to dialog body and according with parameter 'useBin' it will be deleted or recycle. When 'useBin' is true, after close the dialog, the element will be sent to a '#recycle-bin' container and all events will be preserved. If false, it will be deleted and it will not be possible to reuse de same element.

Confirm


eModal.prompt(), do you whant prompt a question? A simple form is ready!


Confirm modal

Is this dialog cool?

How to use

Simple way - Default title
var message = "Wello World!";
                         
eModal.alert(message, null, callback);
Simple way - Custom title
var url = "Wello World!";
var title = "Alert Title";
                         
eModal.ajax(message, title, callback);
Tip: You can use change the default label name! { label: 'Yes' | 'True'| 'OK' }
Message using alternative lable
var options = {
        message: "The famouse question?",
        title: 'Header title',
        size: 'sm',
        callback: function(result) { result ? doActionTrue(result) : doActionFalse(); },
        subtitle: 'smaller text header',
        label: "True"   // use the possitive lable as key
        ...
    };
                 
eModal.prompt(options);
The callback is a mandatory argument, if it is not provided, the dialog will open, however, when on hidden, it will throw an exception. Callback event will retrive a boolean argument as the standard javascritp confirm.
